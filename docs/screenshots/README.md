# Screenshots

Required for the laboratory assignment.

## Per member

Create a folder named after yourself:

```
docs/screenshots/
├── alice/
│   ├── course-progress.png
│   └── sepolia-deploy.png
├── bob/
│   └── ...
```

## What to capture

1. **Course progress** — dashboard showing completed lessons.
2. **Remix** — successful compile (and deploy when ready).
3. **Sepolia** — transaction or contract on a block explorer (when deployed).

Use PNG or JPG. Keep file names descriptive.
