# Course materials

Place Alchemy University or Cyfrin Updraft artifacts here.

## Suggested structure

```
course-materials/
├── notes/           # Per-lesson summaries (markdown)
├── examples/        # .sol files from course exercises
└── links.md         # Bookmarked lesson URLs
```

## Who uploads what

| Folder | Owner | Content |
|--------|-------|---------|
| `notes/` | Rotating per lesson | Short summary + key concepts |
| `examples/` | Per exercise author | Copy of course contract from Remix |
| `links.md` | Member 5 | Index of all lesson links |

Do not commit private keys, API secrets, or `.env` files.
